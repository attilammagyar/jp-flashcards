Particles
=========
A に B で {行く|いく} = B で A に {行く|いく}
A と B と C = A, B and C (and nothing else)
A や B や C など = A, B, C, etc.
と: complete list
や: incomplete list
へ = to, towards, pronounced as え
わ at end of sentence = women make informal statement softer or emotional, men may use it after ~ます/です
の at end of sentence = informal version of の/ん + です (の/ん: female/question, のだ/んだ: male)
とか links representative examples of a class (A and B, and other things)
{何で|なんで}: colloquial "why", {何|なに}で: by what, with what?
か in casual speech = rhetorical question, sarcasm, true or not
か at end of clause = talking about a question
なんか = filler, "like" (casual, from {何|なに} か)
し at end of clause = multiple reasons, etc.

Quotes
======
「」と = direct quote, ~と/~だと = interpreted quote (casual: ~って, following particles can be dropped)
~ と {言う|いう} = called ~ (define; casual: ~って, following particles can be dropped)
~ と {言う|いう} か = rephrase a conclusion / or rather ~ (often casual)
~ て = ~って, except for things actually said/heard
ゆう = {言う|いう} (casual)
つ, つう = rough slang for と {言う|いう}
ってば, ったら = slang (from conditional と {言う|いう}), "I told you a million times!"

Question words
==============
{誰か|だれか} = somebody, {何か|なにか} = something, {何時か|いつか} = sometime, {何処か|どこか} = somewhere, どれか = a certain one from many
{誰も|だれも} = nobody (everybody), {何も|なにも} = nothing (neg.), {何時も|いつも} = always, {何処も|どこも} = everywhere, どれも = any and all
{誰でも|だれでも} = anybody, {何でも|なんでも} = anything, {何時でも|いつでも} = anytime, {何処でも|どこでも} = anywhere, どれでも = whichever

Adjectives
==========
な-adj. + で = adj. and ...
な adj. + に = adverb
い adj. - い + く = adverb
adj. stem/verb stem + そう + です = looks/appears/seems like ~ (personal opinion)
stem of adj. or continuative base of verb + ~{過ぎる|すぎる} = too much ~, over~
くなかった = past negative of い adj.
かった = past tense of い adj.
くない = negative of い adj

Become
======
い adj. - い + く + {成る|なる} = become ~
noun/な adj. + に + {成る|なる} = become ~
verbう + {様|よう} + に + {成る|なる} = to come to be that ~/to become (able to) ~

Location
========
{所|ところ、とこ} = physical or abstract location, part, point
{場所|ばしょ} = physical location
{物|もの}{足りない|たりない}, {物|もの}{足りません|たりません} = be not enough, be not much of a challenge, be missing something

Size, amount
============
{大きい|おおきい} = physically big, {大きな|おおきな} = physical or abstract size (noun modifier, only before nouns)
{小さい|ちいさい} = い-adj., {小さな|ちいさな} = noun modifier, only before nouns

得意, 苦手, 上手, 下手
===================
{上手|じょうず}/{下手|へた}: practical skills, crafts, sports, arts, etc.
{得意|とくい}/{苦手|にがて}: talents, academic/professional skills, strong/weak points
{得意|とくい} is more modest than {上手|じょうず}

Comparatives
============
A は Bより X = Bより A は X = A is X'er than B
Noun-A の {方|ほう} が (Noun-B より) X = Noun-A is X'er (than Noun-B, Noun-B より can be omitted when known)
Verb-A-plain {方|ほう} が Verb-B-plain-nonpast より X = Doing A is X'er than doing B

方がいい
====
な-adj. + な + {方|ほう} + が + いい = ~ is better (than ...)

Because
=======
から = because (verb-plain/~ます + から, ~い/~いです + から, noun/な-adj. + だ/です + から)
の で = because (verb-plain + の で, ~い/~いです + の で, noun/な-adj. + な + の で) (ん で = casual の で)
when answering {何故|なぜ} and どうして, から is more common than の で
verb-plain/い-adj. + の/ん + です = justification, explanation (softer than から)
noun/な-adj. + な + の/ん + です = justification, explanation (softer than から)

Contradictions
==============
の に = despite (verb-plain + の に, ~い/~いです + の に, noun/な-adj. + な + の に)
but: けれども > けれど > けど; ~ます/です --> が; plain --> けど (not always contradiction)

Nouns
=====
noun + に + ついて = about noun
noun + とか + noun + とか = representative examples of a class
noun + の + {頃|ころ} = when ~, as a ~
noun + の + {時|とき} = when ~, as a ~
nounA + と + nounB + の + {間|あいだ} + に = between A and B

Nominalization
==============
verbう + こと = doing ~ (nominalize)
verb stem + {方|かた} = how to do verb (verb --> noun)
noun を verb --> noun の verb stem + {方|かた}
verbう/い-adj. + の = doing ~, (the) ~ one/thing, (clause --> generic noun)
な-adj. + な + の = (the) ~ one/thing, (clause --> generic noun)
verbうこと/noun + に + する = (I have decided I will do) ~
verbうこと/noun + に + した = (I have decided to) ~
verbうこと/noun + に + して + いる = (I make it a rule to) ~
verbうこと + に + {成る|なる}/{成った|なった} = arranged to ~, has been decided to ~ (external decision, less assertive than w/ する)
verbうこと + に + なって + いる = rules, customs, how things are supposed to be done
い-adj. - い + さ = ~ness
な-adj. + さ = ~ness

Potential
=========
noun + できる = can (do) noun (or request, request permission)
{有り得る|ありえる、ありうる} = can possibly exist

ため
==
noun A + の + ため + の + noun B = noun B (is) for the benefit of noun A
noun + の + ため + に + verb = do verb for the benefit of / in order to noun
verb A (dictionary form) + ため + に + verb B = do verb B in order to verb A

欲しい
===
noun + が + {欲しい|ほしい} = (I) want a noun
noun が {欲しがる|ほしがる} = someone else wants ~

未だ, もう
======
{未だ|まだ} + verbて + いません = not done ~ yet, not doing ~ yet
{未だ|まだ} + verbて + います = still doing ~ (incomplete action)
{未だ|まだ}/もう + place に + います/あります = still/already being at ~
もう + verb-past = already completed doing ~
もう + verb-present-negative = no longer doing ~ (complete action)
もう + verbて + いません = discontinued habitual action (no longer)

During/while
============
verb stem + ながら + main verb (present/past) = do/did main verb while doing verb, subject must be the same
(PersonA + が) (verbて +) いる + {間|あいだ} (+ PersonB + は) ... = throughout the whole time that A does/did ~, B does/did ~
(PersonA + が) (verbて +) いる + {間|あいだ} + に (+ PersonB + は) ... = at some point during A does/did ~, B does/did ~
Person + は + (verbて +) いる + {間|あいだ} + に = during A does/did ~, (he/she) does/did ~
verbない + {内|うち} + に = before ~, while not ~

If/when
=======
verb-plain/い-adj. + と / noun/な-adj. + だ + と = if (natural consequence)
verb-plain/noun/い-adj./な-adj. + なら = if (given that X, Y will happen - contextual condition)
verbえば/~い-->ければ/nounであれば = if (general conditional; focus on condition)
(もし +) verbたら (ば) = if/when ~, (then) ~ (もし = if) (focus on result, ~たらば = more formal)
verbたら + past = surprise, unexpected result
verbう/いadj/なadj/noun + かどうか = whether/if

Must
====
verbない - ない + なければ + ならない/いけない, verbない - ない + なくて + は + ならない/いけない = I must ~, social obligations (never advice/request) (ならない ~ rules, いけない ~ 1 person)
noun/な-adj. + で + なければ + ならない/いけない, noun/な-adj. + で + なくて + は + ならない/いけない = I must be ~, social obligations (never advice/request) (ならない ~ rules, いけない ~ 1 person)
noun/な-adj. + で + ない + と + いけない = I must be ~, social obligations (never advice/request) (ならない ~ rules, いけない ~ 1 person)
spoken: なくては --> なくちゃ, なければ --> なくや
と conditional: can imply {駄目|だめ}/いけない/ならない (spoken casual)
negative conditional + {駄目|だめ}/いけない/ならない = must,  have to

Give/receive
============
X は/が Y に/から Z を {貰う|もらう} = X receives Z from Y
X は/が Y に/から verbて {貰う|もらう} = X (asked and) got Y do do ~ (Y != {私|わたし})
X は/が Y から Z を {貰う|もらう} = X(in) receives Z from Y (school, workplace, organization, etc.)
X は/が Y から verbて {貰う|もらう} = X(in) has Y (school, workplace, organization, etc.) to do ~

X は/が Y に Z を {呉れる|くれる} = X(out) gives Z to Y(in)
X は/が (Y を) verbて {呉れる|くれる} = X(out) does ~ for Y(in)
verbて + {呉れる|くれる} = a favor done by equals/subordinates

X は/が Y に Z を {上げる|あげる} = X(out/in) gives Z to Y(out) (never to superiors)
X は/が (Y を) verbて {上げる|あげる} = X(in) does ~ for Y(out) (never to superiors)

X は/が Y に Z を {遣る|やる} = X gives Z to Y (animal, plant)
X は/が Y に verbて {遣る|やる} = X does ~ for Y (animal, plant)

Verbs
=====

transitive verb: active agent acting directly on an object (を)
intransitive verb: never has direct object (を) except for motion verbs

Verbs + particles
-----------------
noun + で + verb = do verb with/using/at noun (place of action or by-means-of)
(place + へ/に) + verb stem + に + {行く|いく} = go to place to do verb

Plain
-----
verbう + {前|まえ} に = before doing ~
verbう + つもり = intend to ~
verbう + {気|き} = feel like doing ~, willing to do ~ (やる{気|き})
verbう + とか + verbう + とか + する = representative examples of a class
verbう + noun = the ~ing noun
verbう + {時|とき} = when doing ~
verbう + でしょう/だろう = probably (will) do ~
verbう + ん + じゃない + でしょう + か = politeness
verbう + か + も + しれません/しれない = might do ~, possibly (will) do ~
verbう + {様|よう} + に + {成る|なる} = change in habitual action
verbう + こと + が + ある = sometimes/occasionally ~
verbう + {様|よう} + に + する = to make an effort to ~
verbう + な = negative command (mostly in fiction only)

Stem
----
verb stem + たい = want to ~
verb stem + たい = want to ~ (verb --> い adj.)
verb-stem + verb = compound verb (action + qualifier)
verb-stem + なさい = imperative (casual: verb-stem + な)
verb-stem + {始める|はじめる} = start doing ~
verb-stem + {出す|だす} = burst out doing ~, start doing ~
verb-stem + {終わる|おわる} = finish doing ~
verb-stem + {続ける|つづける} = continue doing ~

~たい
----
~たい is usually used only in first person or in questions for the second person
~たい: may use が instead of を
~たい + negative: if topic is not mentioned then は can be used instead of が
verbたい -> verbたがる = someone else wants to do ~

て-form
------
verb-A-て verb-B = do verb-A (and) then do verb-B, tense is implied by last verb in the sequence
verbて + いる = be (in the state of) doing ~, emphasize regular habitual action, past action with still existing result (casual speech: ~ている --> ~てる)
motion-verbて + いる = (temporarily) moved and is still there (never continuous present!) (casual speech: ~ている --> ~てる)
verbて + いました = was (in the state of) doing ~, emphasize regular habitual action
verbて + いる + {内|うち} = while doing ~ (unspecified time period)
verbて + みる = try doing ~
verbて + から = after doing ~, since doing ~
verbて + は + いけません = doing verb is not allowed
verbて + は + {駄目|だめ} = must not do verb
verbて + ください = request
verbて + も + いい/{構わない|かまわない} = doing ~ is allowed, (you) may do ~, doing ~ is no problem
verbて + {仕舞う|しまう} = finish doing ~, go and ~ (sometimes regretful; spoken: ~て{仕舞う|しまう} --> ~ちゃう, ~で{仕舞う|しまう} --> ~じゃう)
(A に) verbて + {欲しい|ほしい} = I want (A) to do verb (for me)
verbて + {置く|おく} = leave as it is, do something in advance (casual speech: ~ておく --> ~とく)
verbて {欲しがる|ほしがる} = someone else wants ~ to be done for her/him
verbて + ある = resultant state after completed action (often を --> も/は), (in preparation for sg. else)
verbて + motion ({行く|いく}, {来る|くる}) = direction/orientation of action (physical, time, etc.)
verbて + {呉れ|くれ} = masculin, commanding request
verbて + {頂戴|ちょうだい} = request, please ~ (casual, childish, feminine)

た-form
-------
verbた + こと + が + ある = have done ~ (haven't done ~, have never done ~), past experiences
verbた + {後|あと} + で = after (finished) doing ~
verb-A-た + り, verb-B-た + り, ... + する= (sometimes) do A, (sometimes) do B, ...; representative list of actions in random order
verbた + {時|とき} = when did/have done ~
verbた + か + も + しれません/しれない = might have done ~
verbた + でしょう/だろう = probably did ~
verbた + {方|ほう} + が + いい = you'd better do ~, you should do ~
verbた + {儘|まま} / {儘|まま}だ / {儘|まま}のnoun = remain ~, leaving ~ as is

おう/よう form
------------
verbおう/よう = talking about/to self, I _will_, shall I?
verbおう/よう + と + {思う|おもう} = I think I _will_ ~
verbおう/よう + と + {思って|おもって} + いる = I think someone else will ~, I/someone else is thinking of doing ~
verbおう/よう + と + する = attempt to ~, struggle with ~, tried ~ but failed (can use other verb than する)
verbおう/よう + と + して + いる = to be about to do ~

~ない
----
verbない + で + {下さい|ください} / {呉れる|くれる} = please don't ~ / could (you) not ~
verbない + で + X = (did/happened) X without (doing) ~
verbない + で + {欲しい|ほしい} + です = (I) want (you) not to ~
verbない - い + くて + も + いい + です = don't have to ~
verbない + {方|ほう} + が + いい + です = shouldn't ~
verbない + こと + が + ある = sometimes/occasionally not ~

~える/~られる
--------

verbえる: を --> は/が
verbえる + {様|よう} + に + {成る|なる} = become able to ~

~える/ろ
-----
verbえ/ろ = command (mostly in fiction only)
